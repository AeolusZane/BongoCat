# 精灵宠物素材说明

## 目录结构

```
sprite-pet/
├── pet.json       # 精灵表配置（已创建）
├── idle.png       # 待机状态 - 3帧
├── typing.png     # 打字状态 - 4帧
├── sleep.png      # 睡觉状态 - 2帧
├── happy.png      # 开心状态 - 3帧
├── drag.png       # 拖拽状态 - 2帧
└── click.png      # 点击状态 - 2帧
```

## 素材规格

- **帧尺寸**: 128×128 像素（可在 pet.json 中修改）
- **格式**: PNG，透明背景
- **布局**: 水平排列的精灵表（Sprite Sheet）

## 精灵表格式示例

以 `idle.png`（3帧）为例：

```
┌──────────┬──────────┬──────────┐
│  帧 1    │  帧 2    │  帧 3    │
│ 128×128  │ 128×128  │ 128×128  │
│ 正常姿势  │ 眨眼     │ 睁眼     │
└──────────┴──────────┴──────────┘
         总宽度: 384px
```

## 状态说明

| 状态 | 触发条件 | 动画行为 |
|------|----------|----------|
| IDLE | 无操作时 | 循环播放，30秒后进入 SLEEP |
| TYPING | 键盘/鼠标/手柄活动 | 循环播放，停止活动后回到 IDLE |
| SLEEP | 无操作 30 秒 | 循环播放，任意操作唤醒 |
| HAPPY | 外部事件触发 | 播放一次后回到 IDLE |
| DRAG | 窗口拖拽中 | 循环播放 |
| CLICK | 窗口点击 | 播放一次后回到 IDLE |

## 使用之前生成的素材

之前生成的概念图（桌面宠物_*.png）需要：

1. **裁剪**: 从 16:9 的整图中裁剪出每个状态的精灵表
2. **调整尺寸**: 缩放到 128×128 像素每帧
3. **重命名**: 按上述文件名保存

或者使用像素画工具（如 Aseprite、Piskel）重新绘制，参考概念图的风格。

## 修改配置

编辑 `pet.json` 可以调整：
- `frameWidth` / `frameHeight`: 帧尺寸
- `frames`: 每个状态的帧数
- `fps`: 动画帧率
- `loop`: 是否循环播放
